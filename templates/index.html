Welcome to the AlmostThere demo application.


This application shows the functional aspects of the connection between AT data and Almost There's backend. It uses the information currently provided through Auckland Transport's API.
  1. Click on Get Started
  2. Enter the stop number that you want to start from (e.g. 0026).
  3. This will display a list of routes that will arrive at your chosen stop.
  4. Select a route to find when the next bus will arrive
